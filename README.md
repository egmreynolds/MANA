# MANA
Modelling Association of Non-Additive effects

## Overview
MANA aims to provide functions to run non-additive GWAS models.

## System Requirements
### Julia Requirements:
  Julia 1.1.0
### Julia dependencies
```
BEDFiles
JWAS
BGZFStreams
DataFrames
JuliaDB
IndexedTables
OnlineStats
```

## Installation Guide
Clone repository:
```git clone ${repositoryURL}```

Add to julia script:
```include("MANA.jl")```

Install using Julia package manager is in progress
